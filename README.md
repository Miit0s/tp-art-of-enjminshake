### Tout d'abord, les contrôles
Pour les claviers/souris : 
- Q/D : Gauche/Droite
- Space : Sauter
- Clic gauche : Tirer

Pour les manettes : 
- Joystick gauche : Gauche/Droite
- Bouton du bas : Sauter
- Bouton R1 : Tirer

### Ensuite, le gameplay
Vous pouvez : 
- Tirer sur les monstres
- Tuer les monstres (doivent être touchés plusieurs fois)
- Laissez votre drone faire tout le travail (il tire des missiles à tête chercheuse toutes les X secondes sur un monstre aléatoire. S'il n'y a plus de monstre en vie, il va tirer vers le ciel)
- Créez votre propre niveau en ouvrant le drop down "Level Editor", puis en cliquant sur le bouton "Open Level Editor". Vous pourrez ensuite choisir votre "Outils" en cliquant sur les boutons et modifier le niveau en cliquant sur la grille. Enfin, cliquer sur "Apply Change" pour appliquer les changements. Ils seront sauvegardés dans un dossier nommé "Level Save".

### Concernant le TP
Parmi toutes les features demandées, elles ont toutes été implémentées, à savoir : 
- Créer une entité qui sera le main char avec déplacement basiques et saut
- Éditeur ingame avec imgui pour faire les murs et les ennemis
- Load / Save sur disque
- Des ennemis avec des IA à moustache basiques !
- Death ray - laser shot - Bresenham
- Support manette
- Homing magic missile
- Pet Drone
- Screenshake
- Muzzle fire
- Recul

Pas de features bonus, mais quand même quelque folie : 
- Création d'un système de tween qui gère entre autres : l'effet de hit et de mort des monstres, l'explosion du missile, le shoot du laser et le muzzle
- Il y a des sprites (omg)
