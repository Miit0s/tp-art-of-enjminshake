### Tout d'abord, les contrôleurs
Pour les claviers/souris : 
- Q/D : Gauche/Droite
- Space : Sauter
- Clic gauche : Tirer

Pour les manettes : 
- Joystick gauche : Gauche/Droite
- Bouton du bas : Sauter
- Bouton R1 : Tirer
